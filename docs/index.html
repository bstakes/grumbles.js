Grumbles.js

Detect when your users are upset.

Introduction

Grumbles.js is a very simple library which is able to determine when your user is upset by listening for when they grumble (read: shout swear words). It works via the experimental SpeechRecognition api currently available in the latest version of Chrome.

Installation

NPM


npm i -S grumbles

CDN


<script src="https://unpkg.com/grumbles@1.0.0-alpha.2"></script>

Usage


// The grumbles package main export is a function which takes two
//   arguments:
//   1. A function to call when a grumble is detected.
//   2. An optional string to set the language for
//      SpeechRecognition (defaults to 'en')

const stop = grumbles(({ word, context, time }) => {
  // The function is passed an object containing the word which
  // was grumbled as well as the context (full phrase containing
  // the grumble). You also get a JavaScript timestamp indicating
  // when the grumble was grumbled.
  // At this point you can do whatever you want. Like log the grumble
  // so you can measure grumbles / minute. Or open up a help dialog.
});

// The grumbles function returns a function that will stop listening.
// Call this if you ever want to stop listening.
stop();
